How To Create Your First JS Library

If you look online, Chances are you probably will never find a good source of information about how to make a JS(javascript) Library. This Tutorial will show you how to make a library, or at least explain how they work.

Creating Your Library

First, for your Library to be accessable all over your page, not just in its script, create a new Object Like This:

What is an object?

An object is a variable that can store multiple attributes.


Window.LibraryName = {
};

Now we add our subcatagories. I recommend using subcatagories like css, html, file, etc. to define differnet types of functions in our javascript library. To add a subcatagory, simply create a new object inside of the one you have already created.

Window.LibraryName = {
css = {},
};

Please note that you can also create "sub-subcatagories" by doing the same-adding another object inside of your initial subcatagory, like this:

Window.LibraryName = {
css = {
mySubCatagory = {},
},
};

Another Note: I will not show the catagory, "mySubCatagory" any farther as it was a one time example.


Now, finally, add your functions. They can pretty much do anything, from insert files to your html, to style elements. Insert a function by making it as a variable inside of your subcatagory. Here is an example with a color function:

What is a function?

A Function is a variable that can store a set of actions and take a set of perameters that can be called later.


Window.LibraryName = {
css = {
setcolor: function(id,color){
this.id = id;
this.color = color;
document.getElementById(id).style.color = color;
},
},
};

One question I did have and you that you will probably have too is,"how do I return stuff with my functions?" The answer is simple. Using return before the code that stores your desired output will return the output, for example:

Window.LibraryName = {
css = {
setcolor: function(id,color){
document.getElementById(id).style.color = color;
},
getcolor: function(id,attribute){
this.id = id;
this.attribute = attribute;
return window.getComputedStyle(document.getElementById(id, null).getPropertyValue(attribute));
},
},
};
