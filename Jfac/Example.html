Jfac

Jfac is my First Javascript Library, Not too impressive, but I have decided to make a demo page just in case.

Jfac Alpha includes The following:

  • Add CSS to elements with Jfac.css.add(Id/Class,Css);
  • Get CSS Values With Jfac.css.Get(id/Class,attribute);
  • Add other script files with Jfac.file.bring(scriptname,type);
  • Remove other script elements with Jfac.file.remove(scriptname);
  • Get easier Random numbers with Jfac.math.rand(min,max, true/false to determine whole numbers only);
