Forx.js is a simple but cool small library. It enables a shorthand for pure css images.
Example:
  <div c=green w=100 h=100 bw=100 bc=blue bs=solid br=10></div>
  
Basic Shortcuts:
c: color
w: width
h: height
bw: border-width
bc: border-color
bs: border-style
br: border-radius
bo: border
l: left
r: right
b: bottom
t: top
